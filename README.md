# Customer Ticket Analysis System using Python

## Project Description

This project is a Customer Ticket Analysis System built using Python.
It manages customer support tickets and performs basic text analysis on issue descriptions.

The system stores ticket details such as:

* Ticket Number
* Customer Name
* Issue Description
* Priority Level

It also performs cleaning and analysis of text data.


## Features

## Ticket Management

* Display preloaded tickets
* Add new customer tickets

## Data Cleaning

* Remove punctuation from text
* Convert text to lowercase
* Remove extra spaces
* Replace words (e.g., "okay" → "ok")

## Text Analysis

* Count tickets containing specific words (e.g., poor, good, slow)
* Find ticket with longest issue description
* Extract unique words from all tickets
* Sort and display unique words

##  Priority Analysis

* Count number of High, Medium, and Low priority tickets

##  Technologies Used

* Python
* Lists & Dictionary
* String operations
* Regular Expressions (`re` module)

##  Sample Output

* Displays all ticket details
* Shows cleaned issue descriptions
* Displays word analysis results
* Shows priority distribution

##  Key Learnings

* Data cleaning techniques
* Text processing in Python
* Working with dictionaries and lists
* Basic data analysis logic
